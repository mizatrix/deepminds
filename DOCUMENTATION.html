üëë SUPER ADMINSuper Admin Guide

Important: Super Admin uses the same /admin interface as regular Admins. There is no separate Super Admin dashboard. Additional controls appear on the Users page.

Super Admin Exclusive Features

Feature Location Description
Promote to Admin /admin/users Elevate a Student to Admin role. A "Promote" button appears next to eligible users.
Demote to Student /admin/users Remove Admin privileges. A "Demote" button appears next to Admin users.
Full System Control All Admin pages Access to all Admin features without restrictions.

How to Access Super Admin Features

  1. Log in with your Super Admin account
  2. Navigate to /admin (Admin Panel)
  3. Go to Users page from the sidebar
  4. You'll see Promote and Demote buttons visible only to you
Note: Only ONE Super Admin is allowed in the system. The Super Admin cannot be demoted or deleted.

FOR DEVELOPERSDatabase Schema

Users Table
model User {
  id            String   @id
  email         String   @unique
  name          String
  password      String?
  role          String   @default("STUDENT")
  avatar        String?
  faculty       String?
  studentId     String?
  profileCompleted  Boolean @default(false)

model Submission {
  id               String   @id
  studentEmail     String
  title            String
  category         String
  status           String   @default("PENDING")
  points           Int?
  reviewedAt       DateTime?
  reviewedBy       String?
}
